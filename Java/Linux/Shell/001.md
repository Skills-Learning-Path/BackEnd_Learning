# Shell编程

## Shell概述

Shell 是一个命令行解释器，它为用户提供了一个向 Linux 内核发送请求以便运行程序的界面系统级程序，用户可以用 Shell 来启动、挂起、停止甚至是编写一些程序。

![img](res/1.png)

## Shell入门

脚本以`#!/bin/bash`开头，指定解析器

```shell
#!/bin/bash
echo "abc"
```

### 执行方式

1.  直接执行`./shell脚本名.sh`(需要有执行权限)
2.  sh `./shell脚本名.sh`
3.  bash `./shell脚本名.sh`

### 多命令处理

```shell
#!/bin/bash
cd /home/linux/
touch abc.txt
echo "abc" >> abc.txt
```

## Shell变量

### 系统变量

-   `$HOME`：家目录路径
-   `$PWD`：当前目录路径
-   `$SHELL`：当前Shell路径
-   `$USER`：当前用户

```bash
~$ echo $HOME
/home/sxn
 ~$ echo $PWD
/home/sxn
 ~$ echo $SHELL
/bin/bash
 ~$ echo $USER
sxn
```

### 自定义变量

-   基本语法

    1.  定义变量：`变量=值` **等号两边不能留有空格**
    2.  撤销变量：`unset 变量`
    3.  输出变量：`echo $变量`
    4.  声明静态变量： `readonly 变量`，**注意：不能unset静态变量**

    ```bash
     ~$ A=2
     ~$ echo $A
    2
     ~$ echo $A
    ```

-   变量定义规则

    1.  变量名称可以由字母，数字和下划线组成，不能以数字开头，环境变量名建议大写
    2.  等号两侧不能有空格
    3.  在bash中，变量默认类型都是**字符串类型**，无法直接进行数值运算
    4.  变量的值如果有空格，需要使用双引号或单引号括起来
    5.  可把变量提升为全局变量，可供其他shell程序使用 `export 变量`

    ```bash
    ~$ A=1+1
     ~$ echo $A
    1+1
     ~$ B="abc def"
     ~$ echo $B
    abc def
     ~$ export B
    ```

### 特殊变量

-   `$n`

    $n （描述：n为数字，$0代表脚本名称，10以内参数用$1-9表示，10以上的需要用大括号包含，9表示，10以上的需要用大括号包含，9表示，10以上的需要用大括号包含，{10}）

    ```shell
    #!/bin/bash
    echo "$0 $1 $2 $3"
    ```

    ```bash
    bash test.sh abc def rrr
    test.sh abc def rrr
    ```

-   `$#`

    $# (功能描述：获取所有输入参数个数，常用于循环)

    ```shell
    #!/bin/bash
    echo "$0 $1 $2 $3"
    echo $#
    ```

    ```bash
    $ bash test.sh abc def rrr
    test.sh abc def rrr
    3
    ```

-   `$*`和`$@`

    $* （描述：代表命令行中所有的参数，把所有参数看成一个整体）

    $@ （描述：也代表命令行中所有的参数，不过把每个参数区分对待）

    ```shell
    #!/bin/bash
    echo "$0 $1 $2 $3"
    echo "$#"
    echo "$*"
    echo "$@"
    ```

    ```bash
    $ bash test.sh abc def rrr
    test.sh abc def rrr
    3
    abc def rrr
    abc def rrr
    ```

-   `$?`

    $? （描述：最后一次执行命令的状态，0：正确执行）

    ```bash
    $ echo "$?"
    0
    $ sdfdfs
    ^[[Asdfdfs：未找到命令
    $ echo "$?"
    127
    ```

## 运算符

1.  `$((运算式))` 或 `$[运算式]`
2.  expr +,-,\*,/,% 加减乘除取余
    **expr运算符间要有空格**

```bash
$ expr 3 + 2
5
$ expr `expr 2 + 3` \* 4
20
$ s=$[(2+3)*4]
$ echo $s
20
```

## 条件判断

### 基本语法

`[ condition ]` (注意 condition前后要有空格)

### 判断条件

1.  整数比较

    | 符号 | 描述                     |
    | ---- | ------------------------ |
    | -lt  | （less than）小于        |
    | -le  | (less equal) 小于等于    |
    | -eq  | (equal)等于              |
    | -gt  | (greater than) 大于      |
    | -ge  | (greater equal) 大于等于 |
    | -ne  | (not equal) 不等于       |

2.  文件权限判断

    -   -r 有读的权限
    -   -w 有写的权限
    -   -x 有执行的权限

3.  文件类型判断

    -   -f 文件存在并且是一个常规文件
    -   -e 文件存在
    -   -d 文件存在且是一个目录

```bash
$ [ 23 -ge 22 ]
$ echo $?
0
$ [ 23 -le 22 ]
$ echo $?
1
$ [ -w hellowrld.sh ]
$ echo $?
1
$ [ -e /home/linux/datas ]
$ echo $?
0
```

### 多条件判断

`&&`逻辑与

`||`逻辑或

## 流程控制

### if判断

```shell
if [ 条件判断 ]; then 
	程序
fi
```

或者

```shell
if [ 条件判断 ]
then 
	程序
fi
```

注意：

-   条件判断式前后要有空格
-   if后要有空格

```shell
#!/bin/bash
if [ $1 -eq 1 ]
then
	echo "111"
elif [ $1 -eq 2 ]
then
	echo "222"
fi
```

### case语句

```shell
case $变量名 in
值1)
	程序
;;
值2)
	程序
;;
值3)
	程序
;;

......

*)
	程序
esac
```

注意：

-   case行尾必须为in，每一个模式匹配必须以右括号`)`结束
-   双分号`;;`表示命令序列结束
-   最后的`*)`表示默认模式

```shell
#!/bin/bash
case $1 in
1)
	echo "111"
;;
2)
	echo "222"
;;
3)
	echo "333"
;;
esac
```

### for循环

语法1：

```shell
for((初始值;循环控制条件;变量变化))
do
	程序
done
```

```shell
#!/bin/bash
s=0
for((i=1;i<=100;i++))
do
	s=$[$s+$i]	
done
echo $s
```

语法2：

```shell
for 变量 in 值1 值2 值3 ...
do
	程序
done
```

```shell
for i in $*
do
	echo $i
done
```

### while循环

```shell
while [ 条件判断式]
do
	程序
done
```

```shell
#!/bin/bash
s=0
i=1
while [ $i -le 100 ]
do
	s=$[$s + $i]
	i=$[$i + 1]
done
echo $s
```

## 读取控制台输入

`read(选项)(参数)`

选项：

-   -p 指定读取值时的提示符
-   -t 指定读取值时等待的时间（秒）

参数：

-   变量：指定读取值的变量名

```shell
read -t 7 -p "请在7s内输入" NAME
```

## 函数

### 系统函数

1.  basename

    `basename [string / pathname] [suffix] `

    （描述：basename命令会删掉所有的前缀包括最后一个‘/’字符，然后将字符串显示出来）

    选项：`suffix`为后缀，如果被指定basename会将string或pathname中的suffix去掉

    ```bash
    # 方式1
    $ basename /home/linux/abc.txt
    abc.txt
    
    # 方式2
    $ basename /home/linux/abc.txt .txt
    abc
    ```

2.  dirname

    `dirname 文件绝对路径` 

    （描述：从给定的包含绝对路径的文件名中去除文件名（非目录的部分），然后返回剩下的路径（目录的部分））

    ```bash
    $ dirname /home/linux/abc.txt
    /home/linux
    ```

### 自定义函数

语法：

```shell
[ function ] funname[()]
{
	Action:
	[return int;]
}
funname
```

注意：

-   必须在调用前先声明函数
-   函数返回值只能通过`$?`系统变量获得，可以显式加`return`返回，如果不加将以最后一条命令运行结果作为返回值，`return`后限定数值n(0-255)

```shell
#!/bin/bash
function sum(){
	s=0;
	s=$[$1 + $2]
	echo $s
}
read -p "input your param1:" P1
read -p "input your param2:" P2
sum $P1 $P2
```
